Установка и запуск проекта
1) Cкачать Python 3.10
2) Создать виртуальное окружение
3) Скачать все библиотеки из файла requirements.txt
4) Запустить файл manage.py

Если база данных пуста, и нету никаких аккаунтов, то нужно ввести:
python manage.py createsuperuser
